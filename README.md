## 4️⃣ Convolutional Neural Networks & Computer Vision

One of the major milestones in this repository is a custom **CNN-based plant image classifier**.

### 🌿 PlantVillage Multiclass Image Classification

The project includes:

- The PlantVillage image dataset
- **38 plant-health classes**
- **43,444 training images**
- **10,861 validation images**
- Image resizing to `128 × 128`
- RGB image conversion
- Tensor conversion and normalization
- A custom PyTorch `Dataset`
- Batched loading with `DataLoader`
- GPU-aware training
- A custom CNN architecture built from scratch

### CNN architecture components

- Multiple `Conv2d` layers
- ReLU activation
- Batch normalization
- Max pooling
- Flattening
- Fully connected layers
- Dropout regularization
- Multiclass output layer
- Cross-Entropy Loss
- Adam optimization

This project demonstrates an end-to-end computer-vision pipeline—from reading images and creating labels to training a deep convolutional network.

# ⚙️ Running the Notebooks

## 1. Clone the repository

```bash
git clone https://github.com/Saifuddin-Yasir/ML-AI-Learning-Journey.git
cd ML-AI-Learning-Journey
```

## 2. Create a virtual environment

```bash
python -m venv .venv
```

### Windows

```bash
.venv\Scripts\activate
```

### Linux or macOS

```bash
source .venv/bin/activate
```

## 3. Install the main dependencies

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter \
            torch torchvision torchinfo nltk gensim xgboost kagglehub
```

## 4. Start Jupyter Notebook

```bash
jupyter notebook
```

Some notebooks use datasets hosted on Kaggle or uploaded through Google Colab. Dataset paths may need to be updated before running those notebooks locally.
